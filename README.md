7 | git push | git push is used to upload your local commits to a remote repository.
8 | git pull | git pull is used to download changes from a remote repository and merge them into your current branch.
9 | git checkout | git checkout is used to switch branches or restore files in your repository.
10 | git ignore |.gitignore is a file that tells Git which files or folders to ignore so they are not tracked or committed.
